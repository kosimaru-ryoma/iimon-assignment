# フロントエンド課題 C

- VuexまたはReduxを使って左右を別コンポーネントで実装してください
- 右に移動ボタンで左側の一番下の要素を右側の一番下に移動、左に移動ボタンで右側の一番下の要素を左側の一番下に移動するように実装してください
- 指定されている様にスタイルを当てて下さい

## 内容

- React(vite) 18.12.0
- Node 18.13.0
- npm 8.19.3
- Redux: Redux toolkit 1.9.1
- エディタ: VsCode

- 下記コマンド実行後に、http://localhost:5173 にアクセスし内容確認願います。
```
npm install
npm run dev
```
